HTML

HTML stands for Hypertext Markup Language. HTML first developed by Tim Bernes-Lee in the year of 1990.

It is one of the easiest programming languages to learn. HTML is used to create web pages that are displayed on the Internet. Each page contains a relation to other pages called hyperlink. HTML code ensures the proper formatting of texts and images. Without HTML, a browser would not know how to display text as elements or load images or other element which the user can see on the screen. It is called front end of a web page. HTML also provides the basic structure of the page.

CSS

CSS stands for Cascading Style Sheet. Wium Lie is the developer of CSS and its saga starts in 1994.

It is a simple way for adding style (colors, fonts, spacing etc.) to the web document or a web page. In one word it is use to design front end of a web pages. It allows one to adapt the presentation to different types of devices, such as large screen, small screen or printers. We can design our web pages using CSS as we want to look. One can think HTML as the bone and CSS as the skin of a body.

JavaScript

JavaScript was created by Branden Eric in 1995 during his time at Netscape Communication and was known as LiveScript, but Netscape changed it name to JavaScript possibly because of excitement being generated by Java.

JavaScript is a dynamic computer programing language. It lightweight and mostly used as a part of web pages, whose implementations allow client-side script to interact with the user and make dynamic pages. It is an interpreted programing language with object oriented capabilities.

Python

Python is a high-level, general-purpose programming language. It was created by Guido van Rossum and first released in 1991.

Python language construct and object-oriented approach aim to help programmers to write clear, logical code and large-scale projects. It is the one of the most popular programming language now-a-days. Because of its code are simple and easy to learn and understandable than the other programming language. It is mostly used to create the back-end of a website. And Django is an open source Python web framework used for rapid development, pragmatic, maintainable, clean design and secure websites. It is a high-level free web application framework written in Python.
